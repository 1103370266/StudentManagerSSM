## 学生信息管理系统

#### 技术选型

- Spring+SpringMVC+Mybatis+MySQL+easyUI

#### 完成时间

- 时间两周左右可以搞定

#### 项目结构

![](https://github.com/god-jiang/StudentManagerSSM/blob/master/SSM%E5%AD%A6%E7%94%9F%E4%BF%A1%E6%81%AF%E7%AE%A1%E7%90%86%E7%B3%BB%E7%BB%9F.png)

#### 功能

- 增删查改、分页、模糊查询、权限管理、登录拦截

- 管理员列表（增加、查看、修改、删除）
- 年级列表（增加、查看、修改、删除）
- 班级列表（增加、查看、修改、删除）
- 学生列表（修改）

#### 适合人群

- 在校大学生学习开发MVC模式，简单上手
